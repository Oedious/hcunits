Welcome to HCUnits!

About Us

HCUnit is an open-sourced project created by Clay Wood. The code for the site can be found at the HCUnits GitHub repository. Additionally, the Unit data can be found in JSON and XML form.

You may notice that HCUnits has no advertisements. We intend for it to stay that way.

Help

The site is still a work-in-progress. If you'd like to help improve it, please consider making contributions to the HCUnits GitHub repository or contact us using the information below.
